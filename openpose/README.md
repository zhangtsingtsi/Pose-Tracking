
# OpenPose PyTorch

## Installation

```
git clone https://github.com/zhangtsingtsi/openpose-pytorch.git
cd openpose-pytorch
python setup.py install
```

## Running examples
```
cd examples
python image_demo.py
python video_demo.py
```
